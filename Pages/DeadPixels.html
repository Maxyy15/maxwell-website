Dead Pixels

Dead Pixels is a top-down 2D zombie survival game created in Unreal Engine 5. It features waves of enemies, weapon upgrades, and a retro pixel aesthetic.

Features

  • Fully coded player movement and shooting system
  • Randomized spawns and wave progression
  • Upgrade card system that can upgrade specific guns
  • Weapon swap system
  • Pixel-style graphics
